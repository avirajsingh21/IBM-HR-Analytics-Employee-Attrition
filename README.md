# IBM-HR-Analytics-Employee-Attrition
End-to-end HR Analytics project focused on employee attrition analysis and prediction using Python, Machine Learning, and interactive visualizations.

# Business Problem

Employee attrition leads to increased recruitment costs, reduced productivity, and loss of organizational knowledge. This project analyzes employee data to identify factors contributing to attrition and develops a predictive model to help HR teams identify employees at higher risk of leaving.

# Business Objective

Understand attrition drivers

Identify high-risk employees

Build prediction model

support HR decision making
